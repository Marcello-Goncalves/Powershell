# Powershell Profile

Personal Cross-Platform Powershell Profile Intended to Work on Versions 5.1 (Windows Powershell) and Above (PowerShell - pwsh).

## Intended Usage

Import the '.psm1' Module on This Repo From Powershell Profile: 

```powershell
Import-Module -Name "...\Path\To\PSProfile.psm1" -Global
```

The profile file will probably have one of the following names:
- Profile.ps1
- Microsoft.PowerShell_profile.ps1
- Microsoft.VSCode_profile.ps1
